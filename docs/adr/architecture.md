# Architecture 결정 기록

## 도메인별 헥사고날 아키텍처 적용
- **날짜**: 2026-02-10
- **상태**: 채택

### 컨텍스트
프로젝트는 공시 수집 → LLM 분석 → 시장 반응 조회라는 3개의 명확한 도메인 경계를 가진다.
각 도메인이 독립적으로 발전할 수 있어야 하며, 외부 인프라(DART API, LLM API, 증권사 API)에 대한 의존을 핵심 로직과 분리해야 한다.

### 결정
도메인별로 헥사고날(포트 & 어댑터) 레이어 구조를 적용한다.
의존 방향은 항상 바깥 → 안쪽(domain)으로 흐르도록 한다.
도메인 간 직접 의존은 금지하고, 이벤트 또는 application 레이어 간 연동을 사용한다.
