## 사용 가이드 (이 템플릿을 쓰는 법 — 다 채우고 나서 삭제 가능)

1. **빈칸·`<...>`·*기울임 안내문* 을 본인 내용으로 교체**
2. **첫 작성 30분 이내로 끝내기** — 완벽하게 쓰려 하지 않음. 분기마다 다시 봄
3. **Vision은 한 문장으로** — 길어지면 진짜 무엇을 향하는지 흐려짐
4. **다음 도달점은 검증 가능하게** — "잘 만든다"가 아니라 "X가 Y에서 보인다"
5. **Phase는 의존 그래프로 사고** — 무엇이 무엇을 막는지 명확히
6. **너무 먼 미래는 적지 않기** — 분기 단위 시야면 충분
7. **24시간 묵힌 후 다시 읽기** — 위화감 있으면 그게 진짜 신호
